# K-Heroes 프로젝트

> 문화 빅데이터 기반 한국 역사·지역 문화 인터랙티브 시뮬레이션 서비스

---

## 프로젝트 소개

K-Heroes는 한국의 역사, 지역 문화, 인물, 장소 데이터를 기반으로 사용자가 직접 탐험하고 상호작용할 수 있는 인터랙티브 시뮬레이션 서비스입니다.

사용자는 지역별 문화 콘텐츠를 탐색하고, 역사적 사건과 인물을 기반으로 한 시나리오를 경험하며, 한국 문화에 대한 이해를 넓힐 수 있습니다.
